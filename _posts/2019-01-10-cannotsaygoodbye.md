# 或西更宗油比零桑迪须优

柔情密意怎么可缺少
# 由秦妈一怎么合去修

是进是退也好有若狂潮
# 西跟西退呀豪由遇狂求

是痛是爱也好不须发表
# 西痛西埃呀豪巴须发(biu)

曾为你愿意我梦想都不要
# 曾为泥韵一哦梦颂都巴优

流言自此心知不会少
# 捞因记起桑吉巴会修

这段情越是浪漫越美妙
# 节顿清余西龙门余没缪

离别最是吃不消
# 雷比最西嘿巴修

我最不忍看你背向我转面
# 哦最巴引汉内背航哦军米

要走一刻请不必诸多眷恋
# 要走呀哈请巴比句多俊绿

浮沉浪似人潮那会没有思念
# 浮岑龙西引求拿为没由西尼

你我伤心到讲不出再见
# 尼哦桑森到钢巴区再(gin)
